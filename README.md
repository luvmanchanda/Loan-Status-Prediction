**Loan Status Prediction | Classification Machine Learning Project**

Developed a Machine Learning classification project to predict whether a loan application would be approved or rejected based on applicant demographic, financial, and credit-related information using a structured machine learning workflow.

The project involved collecting a real-world Loan Prediction dataset from Kaggle, followed by comprehensive data cleaning, preprocessing, feature encoding, and Exploratory Data Analysis (EDA). Performed both univariate and multivariate analyses to identify patterns, understand feature relationships, and determine the factors that influence loan approval decisions.

The project workflow included:

* Handling missing values and inconsistent data.
* Encoding categorical variables into numerical representations.
* Performing feature selection and preparing the dataset for model training.
* Splitting the dataset into training and testing sets.
* Training a **Support Vector Machine (SVM)** classifier for binary classification.
* Evaluating the model using the **Accuracy Score** to assess predictive performance.

The trained SVM model achieved an **accuracy of 79.86%** on the test dataset, demonstrating its effectiveness in classifying loan applications based on applicant characteristics.

The project was implemented in Python using Google Colab and utilized industry-standard libraries such as NumPy, Pandas, Matplotlib, Seaborn, and Scikit-learn. The complete project source code has been published on GitHub.

Through this project, I gained hands-on experience in data preprocessing, feature encoding, exploratory data analysis, feature selection, supervised machine learning, model training, model evaluation, and solving a real-world binary classification problem using Support Vector Machines.
